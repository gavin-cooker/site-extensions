#### 此插件用于屏蔽pc常用网站的登录限制,广告,各种弹窗,让自己聚焦于信息主体!

#### 使用油猴或第三方插件不放心,所以写一个自己用,同时作为开源项目!

###### 基于[我的插件脚手架](https://github.com/xzz2021/crx-cli)开发


##### 目前适配的平台:

> * 知乎: `去登录.关注.右下角弹窗`
> * csdn: `去广告` `解除代码复制` `解除一键复制` `直接展开博主文章`
> * 掘金: `去广告` `解除一键复制`
